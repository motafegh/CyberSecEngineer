## The Note-Taking Rules

1. **Every command you learn** → add to the relevant cheatsheet in `tools/cheatsheets/`
2. **Every finding** → save in `vulnerability-research/findings/`
3. **Every CTF/lab** → write a dated write-up in `ctf-writeups/`
4. **Weekly** → git commit all notes with a descriptive message
5. **Every tool discovered** → add to your personal tool reference

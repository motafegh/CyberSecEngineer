## The Note-Taking Rules

1. **Every command you learn** → add to the relevant cheatsheet in `learning/cheatsheets/`
2. **Every finding** → save in `learning/research/findings/`
3. **Every CTF/lab** → write a dated write-up in `learning/ctf-writeups/`
4. **Weekly** → git commit all notes with a descriptive message
5. **Every tool discovered** → add to your personal tool reference
